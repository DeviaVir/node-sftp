{String} options.username
Username on remote host.
{String} options.password Optional
Password on remote host, not required if private key is provided.
{String} options.privateKey Optional
Private key for username, not required if password is provided.
{String} options.host Optional, Default: "localhost"
Hostname or IP to connect to. Default: 'localhost'
{Number} options.port Optional, Default: 22
SSH Port number. Default: 22
{Number} options.timeout Optional, Default: 10000
Inactivity timeout in milliseconds. Default: 10000ms
{Boolean} options.autoconnect Optional, Default: true
Connect on instantiation. Default: true
{String} options.exec Optional
Command to be executed right after a connection is established.

callback to invoke right after a connection is established.

  • readFile(filename, encoding, callback)
    Asynchronously reads the entire contents of a file. The callback is passed two arguments (err, data), where data is the contents of the file. If no encoding is specified, then the raw buffer is returned. Example: 
    sftp.readFile("/etc/passwd", function(err, data) {
    if (err)

    console.log(data);
});
